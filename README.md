## Flutter + Strapi Fullstack CRUD Application

`Git Clone`
```
git clone https://github.com/maheshkareeya/flutter-strapi-crud.git
```
`Change Directory`
```
cd flutter-strapi-crud
```

`Change Directory`
```
cd frontend
```

`Run Command`
```
flutter run
```
`Or Open VSCode and`
```
Ctrl/Cmd + F5
```

`Change Directory`
```
cd backend
```

`Run Strapi Command`
```
npm run develop
```

`Check Url`

http://localhost:1337/admin
